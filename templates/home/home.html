Protein Sequence Design from 3D Protein Structures

ProdCoNN 1.0

A web-based tool for real-time protein sequence prediction and analysis. Trained on a very large number of protein structures, the method called ProDCoNN (protein design with CNN), achieved state-of-the-art performance when tested on large numbers of test proteins and benchmark datasets. Applying a nine-layer 3D deep convolutional neural network (CNN), the server predicts the residue type given the 3D structural environment around the CÎ± atom of a residue, which is repeated for each residue of a protein. All protein structures(input) follow the Protein Data Bank (PDB) standard format. The server also serves the information about each residue's probability at a given position and visible images depicting the overall probability matrix for the user's convenience.

LATEST UPDATES

Demo Page: A demo page has been introduced for the convenience of all kind of users to visualize what they will get out of this server even before submitting any query.
