Demos

For all the demos presented here, the following field values were assumed constant:

PDB file: 1ete.pdb
Chain name: A
Prediction begin range: 1
Prediction end range: -1
Model version: 90

The images of the result page differ only on the basis of the "Model name" field. So we are presenting demos on the basis of two different models. But the downloaded/emailed file is same for any valid input.

Demo 3: Downloaded/Emailed Result File:

In the result page, user can download the result in the form of a .txt file. The same file is also sent to the user via email.

Column 1: It is the Input Amino acid which was given by the user in the PDB file field.
Column 2: It is the predicted Amino acid by our model.
Column 3-22: Rest of the 22 columns contains the probability of each amino acid for that particular position.
